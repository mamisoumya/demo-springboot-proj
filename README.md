# demo-springboot-proj

Created a Order Managment web service layer for a Banquet hall.

Functionalities

 Login to staff
 1. login
 2. Signup
 3. Get staff list
 4. provide access to new user
 
 Orders
 1. create order
 2. delete order
 3. edit order
 4. show list of orders
 
 
Tools or Frameworks used

1. Spring Boot
2. Spring MVC
3. JPA
4. Apache Derby database
5. STS IDE.
